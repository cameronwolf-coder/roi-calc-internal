# Brand Guidelines

## Colors

### Primary Palette

| Color | Hex | Usage |
|-------|-----|-------|
| **Truv Blue** | `#2C64E3` | Primary accent, CTAs, links, key highlights |
| **Dark** | `#171717` | Primary text, headings |
| **Gray** | `#3C3C43` | Secondary text, body copy |
| **White** | `#FFFFFF` | Backgrounds |

### Usage Guidelines

- **Truv Blue** is the primary brand color. Use for CTAs, links, and key UI elements.
- **Dark** for all primary text and headings.
- **Gray** for secondary text and supporting content.
- Maintain high contrast for accessibility.

---

## Typography

### Font Family

**Primary:** Gilroy, Inter, sans-serif

### Font Weights

- **500 (Medium)** - Standard body text and UI elements
- **600-700** - Headings and emphasis

### Sizing Scale

Responsive typography using CSS clamp for smooth scaling:
- Headlines scale from mobile to desktop (e.g., 20px to 46px)
- Body text maintains readability across devices

---

## Logo

### File Location
`/assets/images/logo/logo-truv.svg`

### Usage Rules

- Use SVG format for web
- Maintain clear space around logo
- Don't stretch, rotate, or modify proportions
- Preferred placement: top-left header

## Design Patterns

### Visual Style

- **Modern and minimalist** with clean lines
- **Curved transitions** for visual flow
- **High white space** for readability
- **Data-forward** - let numbers and metrics speak

### Interactive Elements

- Hover states: opacity changes and subtle color shifts
- Transitions: 0.3s-0.4s smooth ease timing
- Accordions: animated arrow rotations
- Focus states for accessibility

### Layout

- Mobile-first responsive design
- Primary breakpoint at 640px
- Consistent padding and margins
- Clear visual hierarchy

---

## Imagery Guidelines

### Do

- Use clean, professional imagery
- Show real interfaces and dashboards where possible
- Include data visualizations that reinforce metrics
- Use charts/graphs that are easy to understand

### Don't

- Use generic stock photos
- Over-design with unnecessary graphics
- Include imagery that distracts from the message
- Use low-resolution or pixelated images

---

## Iconography

- Clean, simple line icons preferred
- Consistent stroke weight
- Match Truv Blue for accent icons
- Use sparingly - don't over-icon

---

## Content Formatting

### Headings

Use clear hierarchy:
- **H1** - Page titles
- **H2** - Section headers
- **H3** - Subsection headers
- **H4** - Detail headers

### Numbers and Metrics

- Bold key numbers: **80%**, **$350**, **30 seconds**
- Use tables for comparing metrics
- Lead with the number when possible

### Lists

- Use bullet points for features/benefits
- Use numbered lists for sequences/steps
- Keep items parallel in structure

### Quotes

- Use blockquote formatting
- Always attribute with name, title, company
- Pull impactful customer quotes prominently

---

## Email Formatting

### Structure

1. **Paragraph 1:** REASON + PROBLEM (2 sentences)
2. **Paragraph 2:** VALUE with social proof (2-3 sentences)
3. **Paragraph 3:** CTA (1 sentence)

### Spacing

- Blank line between each paragraph
- No greeting at the start
- No signature block in templates (added by sender)

### Subject Lines

- Maximum 50 characters
- Lowercase, informal style
- No punctuation at end
- No numbers, AI mentions, or buzzwords

---

## Presentation Guidelines

### Slide Design

- Clean white backgrounds
- Truv Blue for accents and highlights
- One key message per slide
- Lead with data/metrics

### Charts and Graphs

- Simple, clear visualizations
- Use Truv Blue for primary data series
- Gray for secondary/comparison data
- Always label clearly

---

## Document Templates

### Customer-Facing

- Lead with value proposition
- Include relevant proof points
- Use customer logos where permitted
- End with clear next step

### Internal

- Focus on clarity and brevity
- Use consistent formatting
- Include source links where applicable
